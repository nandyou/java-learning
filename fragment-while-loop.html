The while Loop

The while loop repeats as long as a condition is true. Use it when the number of iterations is not known in advance.

public class WhileDemo {
public static void main(String[] args) {
 
// ① Basic while loop
int count = 1;
while (count <= 5) {
System.out.println("Iteration: " + count);
count ++ ; // ← must progress toward false!
}
 
// ② Reading until sentinel value
int input = 0;
while (input != -1) {
// process input...
input = getNextValue(); // imaginary method
}
 
// ③ Summing digits of a number
int number = 12345;
int digitSum = 0;
while ( number > 0) {
digitSum += number % 10; // last digit
number /= 10; // chop last digit
}
// digitSum = 15 (1+2+3+4+5)
 
// ④ Infinite loop with break
while ( true ) {
String cmd = readCommand();
if (cmd.equals("quit")) break;
processCommand(cmd);
}
}
}
Loop variable initialisation
The variable must be declared outside the while loop (unlike the for loop).
This also means it's accessible after the loop ends.
Use for when you need the counter scoped to the loop only.
while ( condition )
The condition is evaluated before every iteration.
If it's false on the very first check, the body never runs.
This makes while a "zero-or-more" loop (vs do-while which is "one-or-more").
Progressing the condition
You must change the condition variable inside the loop, or it loops forever.
This is the most common source of infinite loops in Java!
IDE debugger tip: set a breakpoint here to watch the value change.
Sentinel Value
A special value (-1 here) that signals "stop reading".
Common pattern for reading streams of data with an unknown endpoint.
Example: Scanner.nextLine() returning null at end-of-file.
number > 0 as condition
The loop continues while there are digits left to process.
When number becomes 0 after all digits are removed, the loop stops naturally.
Classic algorithmic pattern for digit extraction, base conversion.
% 10 — Extract Last Digit
Remainder when dividing by 10 gives you the rightmost digit.
Then / 10 removes that digit from the number.
Example: 12345 % 10 = 5, then 12345 / 10 = 1234, repeat...
while(true) — Intentional Infinite Loop
A deliberate pattern for event loops, server loops, game loops.
Always pair with a break or return statement inside.
Used in REPLs (read-eval-print loops) and menu-driven programs.
