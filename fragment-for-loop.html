The for Loop

Java has three flavours of for loop: the classic counted loop, the enhanced for-each, and the modern stream-based forEach.

public class ForLoopDemo {
public static void main(String[] args) {
 
// ① Classic for loop
for ( int i = 0 ; i < 5 ; i++ ) {
System.out.println("Count: " + i);
}
 
// ② Count down
for ( int i = 10 ; i >= 1 ; i-- ) {
System.out.print(i + " ");
}
 
// ③ Enhanced for-each (arrays)
String[] colors = {"Red", "Green", "Blue"};
for ( String color : colors) {
System.out.println(color);
}
 
// ④ break and continue
for (int i = 0; i < 10; i++) {
if (i == 3) continue ; // skip 3
if (i == 7) break ; // stop at 7
System.out.print(i + " ");
}
// Output: 0 1 2 4 5 6
 
// ⑤ Nested loops — multiplication table
for (int r = 1; r <= 3; r++) {
for (int c = 1; c <= 3; c++) {
System.out.print((r * c) + "\t");
}
System.out.println();
}
}
}
Initialisation ( int i = 0 )
Runs once before the loop starts. Declares and sets the loop counter.
The variable i is scoped to the loop — not visible outside.
You can declare multiple variables: int i = 0, j = 10
Condition ( i < 5 )
Checked before each iteration. If false, the loop exits.
If condition is true initially and never becomes false → infinite loop!
An empty condition for(;;) means loop forever.
Update ( i++ )
Runs after each iteration body. Typically increments or decrements the counter.
Can be any expression: i += 2, i *= 3, etc.
The order is: init → [check → body → update] → [check...]
Counting Down
Use i-- as update and i >= 1 as condition to iterate in reverse.
Common for processing arrays from end to start.
Example: for (int i = arr.length - 1; i >= 0; i--)
Enhanced for-each
Reads as: "for each String called color in colors".
Cleaner than index-based loops when you don't need the index.
Works on arrays and anything implementing Iterable (List, Set, etc.).
Cannot modify element values or remove items during iteration.
continue
Skips the rest of the current iteration and jumps to the update step.
Execution continues with the next iteration of the loop.
Use to filter out unwanted values cleanly.
break
Immediately exits the loop entirely, regardless of the condition.
Execution jumps to the first statement after the closing brace of the loop.
In nested loops, break only exits the innermost loop.
Nested Loops
A loop inside another loop. The inner loop runs to completion for each iteration of the outer loop.
Total iterations = outer × inner. Here: 3 × 3 = 9 iterations.
Deep nesting (3+ levels) hurts readability — consider extracting to a method.
