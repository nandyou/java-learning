Java Arrays

Arrays store a fixed-size ordered collection of elements of the same type. Indices start at 0.

public class ArrayDemo {
public static void main(String[] args) {
 
// ① Declare and initialise with literal values
int[] scores = { 90, 85, 78, 92, 88 };
 
// ② Declare with 'new' and set values later
String[] names = new String[3];
names[0] = "Alice";
names[1] = "Bob";
names[2] = "Carol";
 
// ③ Access by index
int first = scores[ 0 ]; // 90
int last = scores[scores. length - 1]; // 88
 
// ④ Iterate with for-each
for ( int s : scores) {
System.out.println(s);
}
 
// ⑤ 2D array
int[][] matrix = { {1,2}, {3,4}, {5,6} };
int cell = matrix[1][0]; // 3 (row 1, col 0)
 
// ⑥ Useful utility
import java.util. Arrays;
Arrays.sort(scores); // sorts in-place ascending
}
}
Array Declaration ( int[] )
The [] after the type signals this variable holds an array.
You can also write int scores[] but int[] scores is preferred style.
All elements must be the same type.
new String[3]
Allocates space for exactly 3 String references on the heap.
Slots default to null until you assign values.
Size is fixed at creation — you cannot resize an array.
Zero-based Indexing
The first element is at index 0, last at length - 1.
Accessing scores[5] on a 5-element array throws ArrayIndexOutOfBoundsException!
Always validate index before access in dynamic code.
.length property
Returns the number of elements. Note: it is a field, not a method — no parentheses.
Compare with String's .length() which IS a method call.
Enhanced for-each loop
Iterates over every element automatically — no index management.
The loop variable s gets a copy of each element in turn.
Cannot modify original elements when using for-each on primitives.
2D Array ( int[][] )
An array of arrays — think of it as a grid of rows and columns.
Access with two indices: matrix[row][col].
Rows can have different lengths (jagged arrays).
Arrays utility class
java.util.Arrays provides helpful static methods:
sort(), binarySearch(), copyOf(), fill(), toString()
Use Arrays.toString(arr) to print array contents.
